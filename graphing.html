Graphing Calculator

A graphing calculator similar to Desmos. There is currently implementation for basic mathematical functions. I am making this with one of my friends in C++.

Current features
  • Camera movement with arrow keys and zooming with the scroll wheel
  • Number line on the x and y axes
  • Efficient graphing engine
  • Function evaluation with a context-free grammar binary tree
Upcoming features
  • Function input box and parser
  • More function support

Made in C++ using the SFML graphics library.
