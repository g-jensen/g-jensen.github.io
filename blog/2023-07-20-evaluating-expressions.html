Evaluating Expressions

2023-07-20

Today, I created functionality to parse and evaluate prefix expressions in my math language. To parse, I created an ExpressionParser class that only has one public method and a couple private ones. The public method is String[] getTokens(String input) and it converts a string like "(+ 23 9)" into an array of strings like ["(","+","23","9",")"], this way it is easier to evaluate expressions which is what I created the functionality for next.
