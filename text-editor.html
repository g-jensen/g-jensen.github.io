Custom Text Editor 🗒️

A lightweight, work-in-progress text editor designed for programming with an emphasis on not having to use the mouse and having full control over customizability.

Current features
  • Basic movement around the file with the arrow keys
  • Creating and deleting new lines
  • Slicing between lines
  • Inputing any unicode characters
  • File importing and exporting
  • Custom keybinding system
  • Custom theme system
Upcoming features
  • Copy and pasting
  • Code snippit support

Made in C++ using the SFML graphics library.
Github repo link: https://github.com/g-jensen/text-editor
