Aim Trainer Game 🖱️

A 2D game that gives you 30 seconds to click as many rectangles on the screen as possible. When a rectangle is clicked, it reappears at a different location. This was my first application using C++ with SFML.

Made in C++ using the SFML graphics library.
Github repo link: https://github.com/g-jensen/aim-training_v2
