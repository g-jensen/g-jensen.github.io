Set


Set is a card game with 81 unique cards that can be played alone or with friends.

Here's a link to an online version of Set that I made in Clojure(Script).

How To Play

Each card has four attributes with three options each: color (red, green, purple), count (1, 2, 3), shape (oval, diamond, squiggle), and shade (open, striped, open).

In the game of Set, 12 cards are delt in front of everyone and the goal is to find a "set". A set is defined as 3 cards where each of the four attributes are either all the same or all different for each card. For instance, the three cards above make a set because each of the attributes are all different.
