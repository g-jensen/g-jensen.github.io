Music Player

This is a custom music player that I made because I didn't want to use spotify. It works by the user inputting a youtube link and then the program downloads the sound as an .ogg using youtube-dl.

Made in C++ using the SFML graphics library.
Github repo link: https://github.com/g-jensen/music-player
